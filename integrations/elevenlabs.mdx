# ElevenLabs Integration

Skaala uses ElevenLabs' cutting-edge conversational AI technology to power natural, human-like phone conversations.

## Overview

ElevenLabs provides:
- **Natural Voice Synthesis**: Ultra-realistic text-to-speech
- **Conversational AI**: Context-aware dialogue management
- **Multi-language Support**: Swedish, Norwegian, English
- **Low Latency**: Real-time voice responses

## Agent Tools

ElevenLabs agents can use Skaala tools during conversations:

```json
{
  "tools": [
    {
      "name": "check_availability",
      "description": "Check calendar availability for booking",
      "parameters": {
        "date": "2025-01-15",
        "service_id": "svc_abc123"
      }
    },
    {
      "name": "create_booking",
      "description": "Book appointment for customer",
      "parameters": {
        "service_id": "svc_abc123",
        "start_time": "2025-01-15T10:00:00Z",
        "customer_name": "John Doe",
        "customer_phone": "+46701234567"
      }
    }
  ]
}
```

## Advanced Features

### Custom Prompts

Fine-tune agent behavior with custom system prompts:

```yaml
system_prompt: |
  You are a friendly receptionist for a hair salon.
  Always:
  - Greet customers warmly
  - Confirm appointment details before booking
  - Suggest alternative times if unavailable

  Never:
  - Discuss pricing over $500 without manager approval
  - Share other customers' information
  - Make promises about services we don't offer
```

### Conversation Context

Maintain context across multiple interactions:

```javascript
{
  "conversation_memory": true,
  "context_window": 10, // Remember last 10 exchanges
  "customer_data_integration": true // Pull from CRM
}
```
