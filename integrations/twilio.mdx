# Twilio Integration

Skaala uses Twilio for reliable telephony infrastructure and phone number management.

<Info>
Twilio integration is handled automatically by Skaala. No manual configuration needed.
</Info>

## Phone Numbers

### Number Assignment

When you subscribe to Skaala, you'll receive:
- Dedicated phone number in your country
- Automatic call routing to ElevenLabs AI
- SMS capabilities for confirmations and reminders

### Number Types

<Tabs>
  <Tab title="Local Numbers">
    **Recommended for most businesses**

    - Local area code in your city
    - Higher trust and answer rates
    - Lower cost than toll-free
  </Tab>

  <Tab title="Toll-Free Numbers">
    **For national businesses**

    - 020 prefix (Sweden), 800/888/etc (Norway)
    - Free for callers
    - Professional image
  </Tab>

  <Tab title="International Numbers">
    **For global presence**

    - Numbers in multiple countries
    - Route to same AI agent
    - Separate analytics per region
  </Tab>
</Tabs>

## Call Features

### Call Recording

<Warning>
Call recording must comply with local laws. Sweden and Norway require two-party consent.
</Warning>

Enable call recording for quality assurance:

```json
{
  "call_recording": {
    "enabled": true,
    "consent_prompt": "This call will be recorded for quality purposes.",
    "retention_days": 90
  }
}
```

### Call Forwarding

Route calls to backup numbers during emergencies:

```javascript
{
  "forwarding_rules": [
    {
      "condition": "ai_unavailable",
      "destination": "+46701234567",
      "announcement": "Connecting you to our team"
    },
    {
      "condition": "business_closed",
      "destination": "voicemail",
      "greeting": "We're currently closed..."
    }
  ]
}
```

## SMS Integration

Send booking confirmations and reminders via SMS:

```javascript
// Example: Send booking confirmation
await fetch('https://www.skaala.ai/api/v1/sms/send', {
  method: 'POST',
  headers: {
    'X-API-Key': process.env.SKAALA_API_KEY,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    to: '+46701234567',
    message: 'Your appointment is confirmed for Jan 15 at 10:00 AM'
  })
});
```

## Number Porting

Bring your existing business number to Skaala:

<Steps>
  <Step title="Check Eligibility">
    Verify your number can be ported (most landlines and mobile numbers supported)
  </Step>

  <Step title="Submit Request">
    Contact Skaala support with current provider details
  </Step>

  <Step title="Approval & Transfer">
    Current provider approves (typically 2-5 business days)
  </Step>

  <Step title="Cutover">
    Number transfers to Skaala with minimal downtime
  </Step>
</Steps>

## Pricing

Twilio costs are included in your Skaala subscription:

| Feature | Included |
|---------|----------|
| Local phone number | 1 number |
| Incoming calls | Unlimited |
| Outgoing calls (transfers) | Pay-per-minute |
| SMS messages | Included with limits |
